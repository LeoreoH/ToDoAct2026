Calles y avenidas

En una ciudad, las calles y avenidas nos ayudan a ubicar lugares.

Las calles son verticales (números) y las avenidas son horizontales (nombres)

¿Cómo se organizan?

  • Calles: Van de norte a sur (verticales) - tienen números
  • Avenidas: Van de este a oeste (horizontales) - tienen nombres
  • Ejemplo: Calle 5, Calle 6...
  • Ejemplo: Reforma, Insurgentes...

Cuadrícula simple

Una cuadrícula nos ayuda a ubicar lugares con coordenadas.

Las columnas son letras (A, B, C) y las filas son números (1, 2, 3)

¿Cómo funcionan?

  • Columnas: Letras (A, B, C...)
  • Filas: Números (1, 2, 3...)
  • Coordenada: Primero la letra, luego el número

Actividad: Lleva el robot a D4

Recuerda: Para llegar a D4, puedes entrar desde cualquier dirección. El robot puede llegar por la derecha, izquierda, arriba o abajo.

Arrastra el robot desde A1 hasta D4. Dejará su huella 🤖 en cada casilla que visite.

🤖

Direcciones

Las cuatro direcciones principales son: Norte, Sur, Este y Oeste.

Norte ↑ · Sur ↓ · Este → · Oeste ←
Norte = arriba · Sur = abajo · Este = derecha · Oeste = izquierda
