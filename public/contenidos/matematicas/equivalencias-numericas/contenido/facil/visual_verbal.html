¿Qué es una equivalencia?

Una equivalencia significa que distintas formas de escribir un número representan la misma cantidad.
1/2
=
0.5
=
50%

¿Qué nos dice esta equivalencia?

  • 1/2 es una fracción.
  • 0.5 es un decimal.
  • 50% es un porcentaje.
  • Las tres formas representan la misma cantidad.

Una misma cantidad, tres formas

Podemos escribir una cantidad como fracción, decimal o porcentaje.

Fracción

Muestra partes de un todo, como 1/4.

Decimal

Se escribe con punto, como 0.25.

Porcentaje

Se escribe con %, como 25%.

1/4
=
0.25
=
25%

Elige la equivalencia correcta

1/4 = 0.25 = 25%
1/4 = 0.5 = 25%
1/4 = 0.75 = 50%
✅ ¡Correcto! 1/4, 0.25 y 25% representan la misma cantidad.
⚠️ Revisa otra vez. Un cuarto equivale a 0.25 y a 25%.

Lee y relaciona

Algunas equivalencias aparecen muy seguido. Conviene leerlas varias veces para recordarlas mejor.
3/4
=
0.75
=
75%

Pista para leer esta equivalencia

  • 3/4 se lee “tres cuartos”.
  • 0.75 es el decimal equivalente.
  • 75% es el porcentaje equivalente.

¿Qué decimal equivale a 3/4?

0.25
0.75
0.5
0.10
✅ ¡Muy bien! 3/4 = 0.75 = 75%.
⚠️ Recuerda: tres cuartos equivale a 0.75.

Completa la equivalencia

Si una parte de la equivalencia falta, podemos completarla usando la forma que representa la misma cantidad.
1/2
=
0.5
=
50%

Completa: 1/2 = ___ = 50%

0.25
0.5
0.75
0.2
✅ ¡Excelente! La equivalencia correcta es 1/2 = 0.5 = 50%.
⚠️ Piensa en la mitad: 1/2 equivale a 0.5.

Encuentra el error

A veces una equivalencia tiene una parte equivocada. Hay que leer con cuidado para encontrar el error.
1/4
=
0.25
=
25%
3/4
=
0.75
=
25%

¿Cuál equivalencia está mal?

1/4 = 0.25 = 25%
3/4 = 0.75 = 25%
✅ ¡Correcto! 3/4 no equivale a 25%; equivale a 75%.
⚠️ Revisa con cuidado el porcentaje. 3/4 equivale a 75%.

Equivalencias en la vida diaria

Las equivalencias también aparecen en situaciones reales, como repartir comida o colorear partes de una figura.

Ejemplo

Si comes la mitad de una pizza, comiste 1/2. Esa misma cantidad también puede escribirse como 0.5 o 50%.

Si una niña comió la mitad de una barra, ¿qué porcentaje comió?

25%
50%
75%
10%
✅ ¡Muy bien! La mitad equivale a 50%.
⚠️ Mitad significa 1/2, y eso equivale a 50%.
